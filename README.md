# seleniumLatestTest
Selenium 3.3.1 example

-Selenium 3.3.1

-chromedriver 2.29 and Chrome web browser 57.0.2987.133 (64-bit)

-firefox geckodriver-v0.15.0-win64 and Firefox web browser 52.0.2 (64-bit)

-Internet Explorer 11 with IEDriverServer.exe
